Pharmacy Alligation Calculator

A Django-based web application to calculate alligation ratios for pharmacy purposes.  
This app is useful for pharmacists, pharmacy students, or anyone learning solution mixing and concentration calculations.

Features

- Calculate alligation ratios for mixing solutions with different concentrations.
- User-friendly interface with instant results.
- Built using Django for a reliable backend.
- Easy to set up and run locally.


Installation

Follow these steps to run the project locally:

1. Clone the repository
2. Navigate to the project folder
3. Create a virtual environment
4. Activate the virtual environment
5. Install dependencies
6. Run the development server
